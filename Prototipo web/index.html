Acerca de nosotros

Running for live es una web destinada a la venta de entradas para un gran número de eventos de running que se celebran en España. Además también disponemos de nuestra propia tienda de productos para el running así como asesoramiento técnico para mejorar tus entrenamientos y lograr que alcances tus metas.
